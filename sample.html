👋 Hey there! Here's how this page works:

This page uses something called anchor links and ID attributes to create a cool scrolling effect. Think of it like a magic elevator that takes you to different floors of a building!

Here's how it works in simple terms:

  1. Every section has a special name tag called an id. For example, this section has id="age-question"
  2. The buttons below have special links called href="#section-name" that tell the page "I want to go to that section!"
  3. When you click a button, the page scrolls smoothly to the section with the matching ID

It's like having a TV remote control where each button takes you to a different channel. The Middle School button takes you to the #middle-school section, and the High School button takes you to the #high-school section!

Try it out by clicking the buttons below! 👇

How Anchor Links Work

This example demonstrates anchor links with ID-based navigation. Here's how it works:

  1. Create an ID attribute: Add id="section-name" to any HTML element you want to link to
  2. Create a link to the ID: Use <a href="#section-name">Link Text</a> to create a link that jumps to that section
  3. Smooth scrolling: Add scroll-behavior: smooth; in CSS to create a smooth scrolling effect

Examples from this page:

  • <section id="age-question">...</section> - Creates a target section
  • <a href="#middle-school">Middle School</a> - Creates a link to the Middle School section
  • <a href="#high-school">High School</a> - Creates a link to the High School section
  • The "Back to Top" button uses <a href="#age-question" class="back-to-top">↑ Top</a>

This technique allows you to create a single-page application feel without requiring JavaScript.
